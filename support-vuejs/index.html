Apoie o Desenvolvimento do Vue.js

Vue.js é um projeto open source sob licença MIT e completamente gratuito para usar.
Entretanto, a quantidade de esforço necessário para manter e desenvolver novas funcionalidade para o projeto não é sustentável sem suporte financeiro apropriado. Você pode apoiar o desenvolvimento do Vue.js através destes métodos:

Doações Únicas

Nós aceitamos doações através destes canais:


BTC
 BCH
 ETH
 LTC

{{ coins[type].address }}

Ajuda Recorrente

Ajudar de forma recorrente garante vantagens exclusivas, como ter seu nome listado no GitHub do Vue, ou ter o logo de sua companhia exibido neste site.

  • Torne-se apoiador ou patrocinador via Patreon (vai diretamente para suportar o trabalho em tempo integral do Evan You no Vue)

  • Torne-se apoiador ou patrocinador via OpenCollective (vai para um fundo com modelo de gastos transparente para suportar esforços da comunidade e eventos)

Se você está rodando um negócio e utilizando Vue em um produto que gera renda, faz sentido para seu negócio suportar o desenvolvimento do Vue: isso garante que o projeto que seu produto depende se mantenha saudável e ativamente mantido. Isso também pode ajudar sua exposição na comunidade Vue e tornar mais fácil atrair desenvolvedores Vue.

Se você tem um negócio que está construindo core products usando Vue, Evan também está aberto a conversas sobre arranjos personalizados de patrocínio / consultoria. Entre em contato pelo Twitter.

Se você é um usuário individual e tem curtido a produtividade de usar Vue, considere uma doação como um sinal de seu apreço - como pagar um café de vez em quando :)
